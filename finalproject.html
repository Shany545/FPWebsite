Our Objective

To provide EMT responders with a means to aid in case of an emergency or need.

Behind the Scenes

Building the robot

Tools: Robot base (breadboard etc.), speaker, lights, and camera.

Preparation: Wireframing, scrum, and walkthrough.

Testing: Each code is individually run through the robot to check for mistakes. Once completion of each code is confirmed we combine all the parts into one code.

Coding

Initial steps

Motions - when sensor hits an obstacle move backwards and turn directions. Create functions for left, right, forward, and back.

Morse Code - when sensor continues to be interrupted flash HELP. Create functions for H, E,L, and P.

Alarm - when whiskers are touched sound alarm between intervals of help.
